Salty-Kore: Simple programming language using python

>SYNTAX: 
```
variable name_of_variable : value Function ( value );
example :
variable x : 500 !SHOW( x );
```
>PREDEFINED FUNCTIONS:
``` 
!SHOW
!ADD
!REMOVE
!CREATE
```
